Types of outputs in JavaScript

  1. innerHTML property

  2. console.log()

    This is used for debugging and testing of the code at development stage and the output displays on console.

  3. document.write()

    This is to display the content directly on the web page.

    By clicking the display button you can see the output.

  4. window.alert()

    It is used to get an alert message box with the ok button on the botton right of the popped message

  5. window.print()

    It is used to get the print dialouge box which can print the current page.

    By clicking print button you can print the current page.

  6. window.prompt()

    It cann't give the output directly but it can ask the user to prompt and take the input.

    To get user input by prompting click the promt button.

  7. console.error()

    It can be to get error message and can be used within asynchronous functions, promises, or callbacks
    to log errors when they occur, helping trace issues in asynchronous code flows.
    Can be seen in console.
