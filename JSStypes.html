JAVASCRIPT can be embed in 3 three ways :

Internal Javascript:This can be embed in the html code either within head tag or in body tag

External JavaScript:This can be embed in the html code with link of javascript file with the extension of "filename.js"

Inline JavaScript:this can be embed with the tags as a property using keywords like alert or warning messages ,onclick , onhover mouse events.
